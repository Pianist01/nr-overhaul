# nr-overhaul

OVERVIEW

This was project was made for a local church with the intention of establishing an online presence for potential new members. The site provides users with information about the church such as the services it holds each week. Given that the church is located in Santa Ana, it's primary audience are Latinos, which is why all the content on the site is in Spanish.

Features

- Responsive navigation
- Contact Page
- EmailJS integration
- Animated Menu
- Multiple pages

TECHNOLOGIES USED

For this project I leaned heavily on JavaScript and CSS. JavaScript was used to generate content while also creating functionality within the page. CSS was used to style everything else along with the page responsiveness. EmailJS was then used to handle the form submissions users send via the contact page.

What I learned

- EmailJS
- Responsive layouts
- Multi-page architecture
